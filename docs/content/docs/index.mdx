# /spicetify

Current release: `v0.1.0`.

`/spicetify` plans local Spicetify workflows before mutation. It uses the `spicetify-agent`
Python tool to inspect, audit, snapshot, apply, verify, report, and roll back local state.

The docs use synthetic examples only and do not read live Spotify or Spicetify state.
